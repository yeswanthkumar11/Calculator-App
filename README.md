# Calculator-App
The Calculator is a basic web-based application built using HTML, CSS, and JavaScript. It performs standard arithmetic operations such as addition, subtraction, multiplication, division, and percentage calculations. The application features a user-friendly interface with a digital display and clickable buttons. It also supports keyboard input, allowing users to perform calculations using both the mouse and keyboard. The calculator provides real-time input display, a clear button to reset the screen, and a delete button to remove the last entered character.

Technologies Used
HTML5 – Structure of the calculator.
CSS3 – User interface design and responsive styling.
JavaScript – Calculation logic and keyboard event handling.
